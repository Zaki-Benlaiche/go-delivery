Procédure d'installation

Ouvrez Google Chrome
Naviguez vers le site dans Chrome sur votre Android.
Menu du navigateur
Appuyez sur les trois points ⋮ en haut à droite de l'écran.
Ajouter à l'écran d'accueil
Sélectionnez « Ajouter à l'écran d'accueil » dans le menu.
C'est prêt !
L'app apparaît sur votre écran d'accueil comme une app native.
Aucun téléchargement requis
La web app s'installe en quelques secondes depuis Chrome et se met à jour automatiquement.
Scanner pour accéder directement

Pointez l'appareil photo de votre téléphone vers le code pour ouvrir l'application instantanément.
